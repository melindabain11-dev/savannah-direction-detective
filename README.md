# 🌟 Savannah's Direction Detective 🌟

A gentle, joyful web game created especially for **Savannah** (rising 1st grader, neurodivergent, sensitive, gifted feeler) to practice recognizing letters and numbers in their **correct direction** versus mirrored/flipped versions.

Built with deep love and care for her beautiful sensitive heart and gentle homeschooling journey. 💖

## 🎮 How to Play

1. **Choose a set** on the home screen:
   - **Letters** (A–Z)
   - **Numbers** (0–9)
   - **Tricky Ones** (b, d, p, q, 6, 9 and other commonly confused characters)
   - **Everything!** (mix of all)

2. **Look at the big character** in the center.
   - It will be shown either correctly oriented **or** flipped horizontally (like a mirror image).

3. **Tap the big green button** if it looks correct.
   **Tap the big red button** if it has been flipped the wrong way.

4. Receive warm, encouraging feedback every time — with confetti celebrations on correct answers and gentle explanations when needed.

Sessions are short (10 rounds) and pressure-free. Perfect for 5–8 minute focused play.

## 🌈 Why This Game Helps Savannah

- **Directionality practice** — critical for reading, writing, and avoiding letter reversals (b/d, p/q, 6/9 etc.)
- **No timers, no scores pressure** — only celebration of effort and sharp observation
- **Immediate visual feedback** — shows the correct orientation on mistakes so she can learn without frustration
- **Neurodivergent-friendly design** — soft pastel colors, large touch targets, encouraging language, optional keyboard shortcuts for mama
- **Choice-based** — she picks what feels good that day

## 👩‍🏫 For Homeschool Mamas

This tool supports gentle, interest-led learning. Use it as a 10-minute morning warm-up, after a nature walk, or whenever Savannah wants to feel successful and capable.

It was designed to celebrate her gifts while gently strengthening a foundational skill in a way that feels safe and fun.

## 🔧 Tech Details

- Pure HTML + Tailwind CSS + vanilla JavaScript (no build step)
- Fully responsive and works beautifully on tablets/phones
- Works offline once loaded
- Open source — feel free to fork and adapt for other gentle learners!
